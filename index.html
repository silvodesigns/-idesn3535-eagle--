Web Site: polygon.com

Polygon is an online magazine that dedicates to the dissemination of information regarding the video game industry and provides its readers with reviews, previews and guides of the most popular video games that come out

The website is fully responsive and is able to switch its layout depending on the screen size the website is being visited from.

The website starts at the mobile version with a single column layout and it eventually becomes a two column layout in the desktop version. There are some minor differences between each layout but they all can access the same content

I chose this website because of its unique colors, predominat geometric shapes and effective execution of a dynamic and responsive layout
